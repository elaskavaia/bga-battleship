# Battleship: Rules & Variants Reference

Reference document for the standard Battleship rules and known variants. Used as a backlog of ideas for future game options.

## Standard Rules (Hasbro 10×10)

- Board: 10×10 grid per player, columns A–J, rows 1–10.
- Fleet: 5-4-3-3-2 (Aircraft Carrier 5, Battleship 4, Cruiser 3, Submarine 3, Destroyer 2).
- Placement: horizontal or vertical only (never diagonal); ships may touch each other and the edge; never overlap.
- Turn: one shot per turn; opponent announces *hit* / *miss* / *sunk*.
- Win: sink the entire enemy fleet.

## Ship-Placement Variants

| Variant | Touch sides? | Touch corners? | Notes |
|---|---|---|---|
| Standard | yes | yes | Hasbro default |
| No-touch (orthogonal) | **no** | yes | Most common house rule |
| No-touch (strict) | **no** | **no** | Used in Bimaru / Solitaire Battleships |

Edges of the board can always be touched.

## Board Size Variants

- **5×5**: each ship occupies 1 cell, distinguished only by HP; 3 ships per side.
- **8×8**: older/budget editions, smaller fleets.
- **10×10**: classic Hasbro.
- **20×20 "Big Fleet"**: extended Salvo edition; multiple shots per turn.

## Fleet Composition Variants

- **Classic Hasbro (post-2002)**: 5-4-3-3-2
- **Hasbro pre-2002**: 5-4-3-2-2-1-1
- **Big Fleet (20×20)**: 8, 7, 6, 6, 5, 5, 4, 4, 4, 3, 3, 3, 3 (one Battleship, one Battlecruiser, two Cruisers, two Light Cruisers, three Destroyers, four Submarines).
- **Indian variant**: T-shaped 5-cell aircraft carrier, one 5-cell battleship, two 3-cell destroyers, two 2-cell submarines.
- **Tetris-block fleet**: ships are tetromino shapes instead of straight lines; ships cannot touch each other.

## Gameplay Variants

- **Salvo (1931)**: one shot per *surviving* ship per turn; firepower decreases as ships are lost.
- **Multi-shot**: fixed 3 or 5 shots per turn regardless of losses.
- **Moving ships**: each turn, a player may move one ship by one cell before firing.
- **Hex grid (2008 edition)**: hexagonal tiles; each board contains islands acting as obstacles; ships placed only around them and only in the player's half.
- **Hit feedback variants**
  - *Strict*: only "hit" or "miss", sinking not announced.
  - *Coordinate-blind*: only the count of hits/misses is announced for salvo turns.
